# ScalerDSA

DSA problems and solutions from the Scaler Academy Core Curriculum, organized by module and topic.

## Structure

There are 2 modules: `Intermediate` and `Advanced`.
Each module contains topic folders (e.g. `Topic1_Time_Complexity`).
Each topic folder contains two subfolders:
- `Assignment` — in-class assignment solutions
- `Homework` — homework problem solutions

## Modules

- **Intermediate** — Time Complexity, Arrays, Prefix Sum, Sliding Window, Sorting, Bit Manipulation, 2D Matrices, Strings, and more.
- **Advanced** — (to be added)
